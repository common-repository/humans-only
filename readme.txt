Stop your WordPress website from being spammed by bots and restrict it only to humans.

== Description ==

If your WordPress website is getting spammed like crazy and nothing seems to be stopping them, then you need something that helps get rid of them once and for all. That's where this plugin comes in that requires everyone to fill out a CAPTCHA before they can do anything on the WordPress website. 

== Installation ==

1. Upload the `humans-only` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to 'Settings' -> 'Humans Only Settings' to enter your reCAPTCHA keys and enable/disable allow spiders.
